Angular Real-time Notifications with ng-event-bus and .NET 8

This demo shows how to implement a real-time notification system using:

  • Angular frontend
  • ng-event-bus for frontend event management
  • .NET 8 backend
  • SignalR for real-time communication

Click the notification icon in the header to see the notification panel. You can test sending different types of notifications using the buttons in the panel.
